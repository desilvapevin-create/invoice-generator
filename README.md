Invoice Generator

A Python tool that automatically generates clean, professional PDF invoices for small businesses and freelancers.

Preview

![Invoice Preview](output/INV-001.pdf)

Features

- Generates professional PDF invoices instantly
- Automatically calculates totals
- Includes client and business details
- Clean, modern design
- Easy to customize

Built With

- Python
- ReportLab

How to Use

1. Clone the repository
2. Install the required library
3. Edit `sample_data.py` with your business and client details
4. Run the script
5. Find your generated invoice in the `output/` folder
